# GestionImmobiliere

Projet PFA - gestion immobiliere

Stack : React Native + Node.js + SQL Server

## Installation

1. Creer la base `RealEstateManagement` dans SQL Server
2. Executer `database/schema.sql` puis `database/seed-data.sql`
3. Backend : `cd backend` → copier `.env.example` en `.env` → `npm install` → `npm run dev`
4. Mobile : `cd mobile` → `npm install` → `npm run android`
